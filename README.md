# Snake_Python

## Description:
Simple Snake game created with Python with help of open source library PyGame. 

## Requirements:
* Python3
* PyGame

## How to run?
When you have Python 3 and PyGame library installed on your device. Just open containg folder and run it by typing:

```
$ python main.py
```

or

```
$ python3 main.py
```
## Enjoy!!
